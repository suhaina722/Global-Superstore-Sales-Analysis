# Global-Superstore-Sales-Analysis
Project Overview
This project analyzes the Global Superstore dataset to understand business performance across sales, profit, customers, products, and regions using Power BI.

The dashboard provides interactive insights into:
Revenue trends
Profitability analysis
Customer behavior
Regional performance

Tools Used
Microsoft Power BI
Excel / CSV dataset
DAX (Data Analysis Expressions)

KPIs Created
Key Metrics:
Total Sales
Total Profit
Total Orders
Profit Margin

Key Insights
West/East regions show highest sales contribution
Technology category generates strong profit margins
Discounts significantly affect profitability
Corporate segment shows stable performance over time
